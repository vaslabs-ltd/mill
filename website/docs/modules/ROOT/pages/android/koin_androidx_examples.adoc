= Koin Androidx Examples
:page-aliases: koin_androidx_examples.adoc


This page provides an example of how https://github.com/InsertKoinIO/koin/tree/main/examples/androidx-samples[Koin AndroidX Samples] can be built with Mill.

This example includes Roboelectric tests which are configured to run on the JVM and not on an Android device, which is a common setup for Android projects.
